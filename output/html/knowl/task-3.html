Task 3.2.3.a.

Find the relation between the tension \(T\) and weight \(W\) for the system of FigureÂ 1 using scalar components.

Answer
Solution

The forces in this problem are \(\vec{W} = -W\jhat\) and \(\vec{T} = T\jhat\text{,}\) so the corresponding scalar components are

\begin{align*} W_y \amp= -W \amp T_y \amp= T . \end{align*}

Adding scalar components gives,

\begin{gather*} \Sigma F_y = 0\\ W_y + T_y = 0\\ -W + T = 0\\ T = W \end{gather*}

Unsurprisingly, we get the same result.
